# InspiraVerse Feature Deep-Dive

## 📈 Mindset Journey: Quantitative Motivation

The Mindset Journey is more than just a streak counter; it is a behavioral analytics engine designed to visualize personal development.

### Core Logic
1. **Engagement Scoring**: Every interaction (Reflect, Favorite, Share) adds points to the daily `mindset_score`.
2. **Resilience Heatmap**: Visualize activity frequency across a 14-day rolling window to identify patterns of peak motivation or burnout.
3. **Journal Integration**: "The Mindset Mirror" links subjective reflections to objective quote metrics, creating a complete historical timeline of personal wisdom.

## 🎨 Designer Studio: Viral Card Engine

We optimized the Designer Studio for "Aesthetic Performance". 

### Key Capabilities
- **Premium Font Pairings**: Curated typography (Outfit, Cinzel, Montserrat) optimized for mobile readability.
- **Glassmorphic Export**: Real-time rendering of blurs and transparency into static image formats for seamless social sharing.
- **Auto-Branding**: Intelligent watermark placement that ensures user-generated content remains attributed to InspiraVerse while staying visually pleasing.

## 🧠 Curated Psychology Datasets

Unlike generic apps, InspiraVerse uses categorized psychological segments:
- **Resilience**: Stoic and modern endurance quotes.
- **Focus**: Efficiency and intentionality.
- **Mindfulness**: Anxiety reduction and present-moment awareness.
- **Growth**: Ambition and adaptive learning.
